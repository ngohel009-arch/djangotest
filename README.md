# 🏋️ Gym Management System

A complete **Gym Management System** developed using **Python Django** and **SQLite3**. This project helps gym administrators manage members, trainers, membership plans, attendance, payments, workout plans, equipment, enquiries, and feedback through an easy-to-use web interface.

---

## 📌 Project Features

### 👨‍💼 Admin Module
- Admin Login
- Dashboard with Statistics
- Add, Edit & Delete Members
- Add, Edit & Delete Membership Plans
- Add, Edit & Delete Trainers
- Manage Equipment
- Manage Workout Plans
- Mark Member Attendance
- View Payments
- Manage Enquiries
- View Member Feedback
- Logout

---

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Gym-Management-System.git
```

---

### Move into Project Folder

```bash
cd Gym-Management-System
```

---

### Create Virtual Environment

```bash
python -m venv venv
```

---

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

---

### Install Requirements

```bash
pip install -r requirements.txt
```

---

### Run Migrations

```bash
python manage.py makemigrations
```

```bash
python manage.py migrate
```

---

### Run Server

```bash
python manage.py runserver
```

---

Open your browser and visit

```
http://127.0.0.1:8000/
```
